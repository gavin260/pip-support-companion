
# PIP Support Companion Web App

A simple React/Vite web app prototype for ethical PIP self-reporting support.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Deploy to Vercel

1. Create a GitHub repository and upload these files.
2. Go to Vercel and import the repository.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Deploy.

You will get a public URL that you can send to applicants.

## Deploy to Netlify

1. Create a GitHub repository and upload these files.
2. Go to Netlify and add a new site from Git.
3. Build command: `npm run build`.
4. Publish directory: `dist`.
5. Deploy.

## Notes

This prototype stores data only in the applicant's browser local storage. It is not affiliated with DWP and is not legal advice.
